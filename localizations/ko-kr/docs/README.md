# /docs

이 폴더는 세션의 문서와 단계별 콘텐츠를 담습니다.

## 여기에 들어가는 내용

- **Labs/Workshops**: 번호를 매긴 실습으로 정리한 단계별 안내(예: `01-setup/`, `02-first-exercise/`)
- **Demos**: `/src`의 데모 코드를 설명하는 워크스루 문서
- **Breakouts**: 보충 문서, 다이어그램, 참고 자료

## 팁

- 순서를 위해 번호 접두사를 사용하세요: `01-setup/`, `02-exercise/`, `03-wrap-up/`
- 각 하위 폴더는 자체 `README.md` 또는 `index.md`를 둘 수 있습니다
- 필요하다면 이미지는 `assets/` 하위 폴더에 보관하세요
- 세션에 README 외에 별도 문서가 없다면 이 폴더는 자유롭게 삭제하세요
